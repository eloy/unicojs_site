1 Unico, a reactive framework

Unico is a js framework built on top of Facebook React and inspired by Angular. It is suitable for small widget as well as full single page applications.

1.1 Documentation

1.1.1 Installation

Download Unico from the release page on github: https://github.com/harlock/unicojs/releases You can choose one of these versions:

  • unico.core.js: Just the framework. You must download and require dependencies.
  • unico.js: Bundle with unico and its dependencies. Development version.
  • unico.min.js: Production ready build.

1.1.2 Controllers

In order to use Unico as an standalone framework in your application, you should add the tag 'controller' to the container. Controllers are responsible of provide data to the view and execute callbacks on user actions.

1.1.3 Directives

Directives are the bridge between react elements and your code. Directives give you the chance to manipulate the DOM node, add custom listeners, or change the element behavior

  1. Built in directives
    1. if and unless

      These directives control the visibility of the element.

    2. repeat

      This directive let you iterate over arrays and objects.

    3. click

      This directive execute a callback on click events. After execute the callback, it check for changes and update the view.

    4. model

      This directive provide 2 way data binding.

    5. submit
  2. Create custom directives
